Anthropic API ključ

Za AI analizo načrtov in generiranje ponudb potrebujete API ključ.
Dobite ga brezplačno na console.anthropic.com
(nova registracija vključuje ~5 $ kreditov ≈ 100–150 analiz)

✅ API ključ shranjen v napravi
Navodila za postavitev na splet

1. Registracija GitHub
Pojdite na github.com → Sign up → brezplačni račun

2. Nov repozitorij
New repository → ime: ponudba-struzenje → Public → Create repository

3. Naložite datoteko
Add file → Upload files → naložite ponudba-struzenje.html → preimenujte v index.html → Commit changes

4. Vklopite GitHub Pages
Settings → Pages → Source: Deploy from branch → Branch: main → / (root) → Save

5. Vaš naslov aplikacije
https://VASEIME.github.io/ponudba-struzenje

6. API ključ
console.anthropic.com → API Keys → Create Key → prilepite ga v polje zgoraj

7. Nadgradnje
Ko bova dodelala aplikacijo, zamenjate samo datoteko index.html na GitHubu — posodobitev traja minuto.
